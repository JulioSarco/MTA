Proyecto Give me a like


El proyecto Give me a like es un proyecto patrocinado por el ministerio de educacion , es este proyecto tenemos que crear un canal de Youtube y conjuntamente afrontar una serie de retos.

Nº Contenido del reto
0º En este reto teniamos que crear un canal de Youtube
1º En el primer reto teniamos que conseguir hacer control remoto desde Windows,Ubuntu y dispotivo mòvil
2º Esta vez teniamos que enseñar como se instalaba,configuraba y monitorizaba un Debian server con OpenLDAP
3º Como tercer reto teniamos que hacer una serie de videos demostrando nuestro conocimiento de html y además crear un portal web en Github
4º Por ultimo tenemos que crear un raid0, raid1 y raid5 en Windows10 y Ubuntu
